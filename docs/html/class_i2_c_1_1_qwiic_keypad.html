Detailed Description

A class with Singleton interfaces to communicate between the SparkFun Qwiic I2C Keypad and the LPC 804 development board.

◆ Poll()

status_t I2C::QwiicKeypad::Poll ( ) const

Polls the I2C Keypad once with a short delay post execution. If a button has been found to have been pressed since the last invocation of I2C::Keypad::Poll(), all registered callback functions will be invoked with the pressed button index as their parameter.

See also
I2C::Keypad::AddListener(ButtonCallback)
Returns
kStatus_Success if the poll was successful;

kStatus_I2C_UnexpectedState if the button cannot be read;

another error status if there was an internal I2C error
